Seiryo Linux

清涼 — clarity, coolness, volume

Seiryo is a DIY, rolling-release meta-distribution for those who value coherence, choice, and control—without chaos.

Built around koushou, a fast, KDL-native package manager, Seiryo offers multiple init systems, C libraries, and userlands—each a complete, tested system flavor.

  • Flavors: systemd, OpenRC, s6, dinit × glibc, musl
  • Package manager: koushou — daemon-powered, atomic, KDL-based
  • Stable rolling: tested updates, rollback support
  • Desktop-ready: GNOME (patched for non-systemd), KDE, or minimal
  • 100% open: passion project, no corporate backing

Not a replacement. Not a clone. Just a system that fits.
